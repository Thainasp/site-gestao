Este projeto consiste em uma página web simples de acesso ao sistema do Instituto Federal de São Paulo (IFSP). O sistema oferece dois tipos de autenticação:

Acesso como Administrador;
Acesso como Usuário.

Além disso, o sistema possui uma validação básica dos campos de entrada, impedindo que o campo de usuário seja enviado vazio.

O projeto foi desenvolvido como atividade da disciplina de Gestão de Projetos de Software, servindo como exemplo para planejamento e gerenciamento de releases.

Funcionalidades
Implementadas
Tela de login.
Seleção de perfil de acesso:
Administrador.
Usuário.
Validação do campo de usuário.
Bloqueio do envio do formulário quando o campo de usuário estiver vazio.
Regras de Negócio
O campo Usuário é obrigatório.
O sistema não permite prosseguir caso o campo esteja vazio.
O acesso pode ser realizado por dois perfis distintos: Administrador e Usuário.
